Contador

Este é um projeto Javascript onde é implementado um contador.

Os objetos da DOM são manipulados através do Javascript para que reflitam os cliques dos botões do contador, sendo que os valores estão delimitados entre mínimo 0 e no máximo 50.

0

Como desafio do instrutor, foi adicionado o tratamento por eventos (addEventListener) para executar as funções dos botões.
Também foi aplicado um listener keydown para capturar teclas pressionadas no teclado.

para que o contador seja operado com as teclas (+), (-) e (0) para zerar.
